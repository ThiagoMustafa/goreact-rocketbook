# goreact-rocketbook
A simple project using ReactJS to create a similar facebook desing.  This project is part of a traning about ReactJS of company Rocketseat

#### 1 - Clone the project.
#### 2 - Execute command to install dependences:
```npm install```
### 3 - Run
```npm run dev```
### 4 - Access the url
https://localhost:8080
